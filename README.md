# Raumigo – Legal Pages

Static website for the legal documents of the Raumigo iOS app.
Hosted via GitHub Pages.

## Pages

- `index.html` – Overview (German)
- `agb.html` – Terms and Conditions (German)
- `datenschutz.html` – Privacy Policy (German)
- `privacy.html` – Privacy Policy (English)
- `impressum.html` – Legal Notice (German)

## Setup

1. Create a new GitHub repository (e.g., `raumigo-legal`)
2. Push these files to the `main` branch
3. Go to **Settings → Pages** → Source: `main` / `/ (root)`
4. Your pages will be available at `https://YOUR-USERNAME.github.io/raumigo-legal/`
5. Use `https://YOUR-USERNAME.github.io/raumigo-legal/privacy.html` as the Privacy Policy URL in App Store Connect
